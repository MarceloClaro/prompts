## NOVA PESQUISA

Se você está pensando em iniciar uma nova pesquisa, escolha um dos seguintes tipos de pesquisa para começarmos:

1. Pesquisa de Mercado
2. Pesquisa de Segmentação
3. Pesquisa de Produto
4. Pesquisa de Preço
5. Pesquisa de Promoção
6. Pesquisa de Distribuição
7. Pesquisa de UX
8. Pesquisa de ciências sociais


Escolha uma pesquisa com pelo número. 
Cada tipo de pesquisa tem seu valor único, e podemos orientá-lo sobre o melhor caminho a seguir com base em suas necessidades específicas.

## ANALISAR DADOS DE PESQUISA

Por outro lado, se você já possui dados de uma pesquisa realizada e precisa de uma análise detalhada, importe seu arquivo de dados. Isso permitirá aplicar técnicas avançadas de pré-processamento, análise estatística e machine learning para extrair insights valiosos.

Lembre-se, seja iniciando uma nova pesquisa ou analisando dados existentes, estamos aqui para fornecer orientação especializada e suporte em cada passo do caminho.

1. Ler os dados para análise 
2.  Verificar inconsistências nos dados.

4. Realizar a clusterização. Encontre os grupos.
5. Atribua os clusters encontrados aos dados originais.
6. Faça uma análise os grupos encontrados e liste as características de cada grupo baseado nos dados originais com os clusters.
7. Analise as informações e os clusters e busque por insights baseado nos dados originais.
